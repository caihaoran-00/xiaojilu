# 更新日志

## v2.2.0 (2026-02-24)

### 新增
- 自定义事件：用户可在时间点和持续页面自行添加新的事件类型
- 自定义事件管理：支持删除自定义事件（管理/完成切换模式）
- 备注功能：每条记录（时间点和持续）均可添加备注
- 备注建议：同一事件类型的历史备注可复用（localStorage 缓存）
- 动态历史筛选：历史页面的筛选标签根据实际记录动态生成
- 编辑弹窗：修改活动类型时包含自定义 duration 事件
- 后端：新增 `custom_events` 表，支持按家庭存储自定义事件
- 后端：新增 `GET /api/custom-events` 接口
- 后端：新增 `POST /api/custom-events` 接口
- 后端：新增 `DELETE /api/custom-events/:id` 接口
- 后端：新增 `PATCH /api/instant/:id/note` 接口
- 后端：新增 `PATCH /api/duration/:id/note` 接口

### 变更
- 时间点和持续页面的事件按钮改为动态渲染（内置 + 自定义）
- 历史记录筛选标签改为根据实际数据动态生成

## v2.0.0 (2026-02-22)

### 新增
- 多家庭支持：通过密码区分不同家庭，每个家庭数据隔离
- 管理后台：独立页面 `/admin.html`，支持创建/编辑/删除家庭
- 管理后台：支持配置宝宝名字（可选）
- 图片上传：任何记录均可上传图片，客户端自动压缩（最大 1200px，JPEG 质量 0.7）
- 图片预览：点击缩略图可全屏预览，支持删除
- 最近 10 天记录：时间点和持续页面按日期分组展示最近 10 天数据
- 宝宝名字显示：页面标题显示"XX的小记录"
- 后端：`families` 表、数据迁移、级联删除
- 后端：`images` 表、multer 文件上传
- 后端：`GET /api/recent?days=10` 接口

### 变更
- 管理后台从主页分离为独立端点 `/admin.html`
- 主页 `index.html` 不再包含任何管理后台代码
- 认证方式改为基于家庭密码的 token
- README 更新为 v2 文档

## v1.0.0 (2026-02-21)

### 功能
- 时间点记录：一键记录换尿裤、喂奶等一次性事件
- 持续记录：开始/结束计时，自动计算时长（晒太阳、吃奶等）
- 进行中活动：顶部横幅实时显示，任意家庭成员可结束
- 家庭共享：三人（爸爸/妈妈/奶奶）共享数据，15 秒自动刷新
- 历史记录：按日期分组查看，支持按事件类型筛选
- PWA 支持：可添加到手机主屏幕
- 一键部署脚本 `deploy.sh`
